**Дата:** 24 мая 2026
**Страница:** `src/pages/index.astro`
**Статус:** Утверждён

## Текущее состояние

Сайт полностью текстовый — ни одной картинки, пустые поля по бокам на широких экранах, faint gradient'ы которые почти не видны. Контент 720px + 320px в контейнере 1240px = ~110px мёртвого пространства с каждой стороны.

## Что делаем

### 1. Hero-секция — декоративный фон + анимированные фишки

**Задача:** Заполнить пустое пространство вокруг hero-текста.

**Реализация:**
- Gradient-сетка (CSS grid pattern) из тонких линий по бокам — `repeating-linear-gradient`
- Noise-текстура (SVG filter `feTurbulence`) как overlay с opacity 3-5%
- Световые лучи (radial-gradient эллипсы) — 2-3 штуки, разная интенсивность
- Анимированные покерные фишки (3-5 штук) — CSS `@keyframes float`, парят вокруг контента, разные размеры, `opacity: 0.15-0.4`, `pointer-events: none`
- Фишки: gold, purple, black — цвета из палитры ChipStack

**Файлы:**
- `src/components/HeroDecor.astro` — компонент с фоновыми элементами
- `src/styles/animations.css` — keyframes для float, pulse, fade-in

### 2. Секция «Краткий вердикт» — текстура + боковые элементы + иконки

**Задача:** Сделать текстовую секцию визуально насыщенной.

**Реализация:**
- Noise/grain текстура поверх фона секции (повторяется из hero)
- Боковые абстрактные gradient-формы — 2-3 эллипса с `rgba(105, 124, 200, 0.05)` по краям
- SVG-иконки рядом с каждым ключевым абзацем:
  - 💰 Депозит — иконка кошелька/монеты
  - 💸 Вывод — иконка стрелки вывода
  - 🎰 Слоты — иконка слот-машины
  - 🔒 Верификация — иконка щита
  - ⚡ Скорость — иконка молнии
  - 📱 Мобильная — иконка телефона
- Иконки: SVG inline, stroke-based, цвет `var(--color-gold-dim)`, размер 24x24px

**Файлы:**
- SVG иконки встроены в `index.astro` или вынесены в `src/components/icons/`

### 3. Анимации — scroll + hover + микро

**Задача:** Оживить интерактивность без ущерба производительности.

**Реализация:**
- **Scroll-анимации:** IntersectionObserver в `src/scripts/scroll-reveal.js`
  - `.reveal` класс: `opacity: 0; transform: translateY(20px)`
  - При входе в viewport: `opacity: 1; transform: translateY(0); transition: 0.6s ease`
  - Применяется к: секция статей, секция сравнения, быстрые ссылки, карточки
- **Hover на карточках:** уже частично есть, дополнить:
  - `box-shadow` glow при hover (`0 0 20px rgba(247, 203, 106, 0.15)`)
  - Плавный `transition: all 0.25s ease`
- **Микро-анимации:**
  - Кнопки: `transform: scale(1.02)` при hover, `scale(0.98)` при active
  - Animated underline на ссылках: `::after` pseudo-element с `scaleX` transition (уже есть `.link-underline`)
  - Пульсация золотых элементов: subtle `opacity` keyframe 0.8-1.0

**Файлы:**
- `src/scripts/scroll-reveal.js` — IntersectionObserver логика
- `src/styles/animations.css` — все keyframes и transition-классы

### 4. Карточки статей — SVG-иконки + превью

**Задача:** Сделать карточки визуально информативнее.

**Реализация:**
- В `ArticleCard.astro` добавить SVG-иконку по тегу статьи:
  - `moriwin`, `казино` → иконка слот-машины
  - `бонус`, `промокод` → иконка подарка/процента
  - `зеркало` → иконка зеркала/ссылки
  - `криптовалюта`, `токен` → иконка монеты
  - `vpn` → иконка щита
  - `регистрация` → иконка пользователя
  - `вывод` → иконка кошелька
  - по умолчанию → иконка статьи/документа
- Иконка в верхнем левом углу карточки, 32x32px, цвет `var(--color-gold)`
- Превью-картинки: пока не генерируем автоматически, только если пользователь предоставит

**Файлы:**
- `src/components/ArticleCard.astro` — обновлённый компонент
- `src/components/icons/` — SVG иконки (или inline в компоненте)

## Ограничения

- Никаких внешних библиотек (чистый CSS + vanilla JS)
- Все анимации через `transform` и `opacity` (GPU-accelerated)
- `will-change` только на элементах с реальными анимациями
- Мобильная адаптация: количество фишек уменьшается, декор скромнее
- Общий вес новых ассетов < 50KB

## Что НЕ делаем

- Нет тяжёлых фоновых анимаций (performance)
- Нет логотипа MoriWin (независимость издания)
- Нет автоматической генерации превью-картинок (пока)
- Нет video/webm файлов
